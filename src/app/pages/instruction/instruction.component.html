Vous avez été invité(e) à passer un test Vous devrez effectuer cette évaluation sans l'aide d'une tierce personne.

Les importantes Instructions

  • L' examen est composé des questions à choix multiple.
  • Le test est chronométré.La durée total est de 20 minutes.
  • Vous pouvez envoyé accéder à votre examen dans quelles moment vous voulez .
  • Chaque question est notée. Pas de note négative pour les réponses fausses.
  • Toutes les questions sont de type QCM.

  • Cliquer sur Commencer test button pour commencer l'examen
  • Le temps débute au moment où vous cliquez sur le boutton de début
  • Vous ne pouvez plus revenir à l'examen en cas d'interruption à cause de quelles raisons.
  • Vous n'avez plus le droit d'annuler vos réponses une fois vous avez commencer.
  • Cliquez sur le bouton Soumettre le test à la fin de l'examen.
